===== PAGE 11 =====
Prerequisite: NumPy
▪ Q) (Array) What is difference between list and NumPy array?
– Both can concatenate Python data in multiple dimensions.
• List: Heterogeneous data type / NumPy array: Homogeneous data type
– Note) NumPy array is close to a concept of array in C/C++ and Java.
– NumPy array provides
• Advanced indexing (e.g. logical/fancy indexing)
• Advanced slicing (e.g. column/subarray slicing)
• Arithmetic operations (e.g. addition, multiplication)
• Linear algebra (e.g. matrix multiplication, pseudo-inverse)
• Useful utilities and tools (e.g. load()/save(), loadtxt()/savetxt())
